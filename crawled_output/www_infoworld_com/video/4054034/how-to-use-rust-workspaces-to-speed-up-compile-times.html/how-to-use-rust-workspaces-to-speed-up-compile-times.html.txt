OverviewRust projects can have notoriously long compile times. One way to speed up compilation is to split a project into multiple crates in a single "workspace", where each crate can be recompiled independently of the others if they're changed. This video shows how to do that with a simple example project — and how using a workspaced Rust project can also make it easier to manage overall.Register Now
