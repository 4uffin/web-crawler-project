Tutorial de instalação para o Agente de Mapas Mentais
Raw
tutorial-mapas-mentais.md
Instalar programas
Obsidian
Claude Desktop
Bun (ver abaixo como instalar)
Como instalar o Bun
Windows
Abrir Powershell e executar comando
powershell -c "irm bun.sh/install.ps1 | iex"
macOS
Abrir terminal e executar comando
curl -fsSL https://bun.sh/install | bash
Instalar plugins do Obsidian
Abrir configurações
Plugins não oficiais > Ligar os plugins da comunidade
Instalar plugins
Mindmap Nextgen
Local REST API
Nas configurações, habilitar "Enable Non-encrypted (HTTP) Server"
Copiar API Key para ser usada na configuração do Claude
Configurar Claude Desktop
Habilitar modo desenvolvedor
Ajuda > Habilitar modo desenvolvedor > Habilitar
Configurar servidor MCP
Arquivo > Configurações > Desenvolvedor > Editar configuração
Abrir claude_desktop_config.json
Colar configurações (não esqueça de atualizar com a sua chave de API do Obsidian!!)
{
"mcpServers": {
"@fazer-ai/mcp-obsidian": {
"command": "bunx",
"args": [
"@fazer-ai/mcp-obsidian@latest"
],
"env": {
"OBSIDIAN_API_KEY": "<sua api key do obsidian>"
}
},
"youtube-transcript": {
"command": "bunx",
"args": [
"-y",
"@kimtaeyoon83/mcp-server-youtube-transcript"
]
}
}
}
NoteCaso você tenha outros servidores MCP já configurados no seu Claude, basta incluir o objeto "@fazer-ai/mcp-obsidian" dentro de mcpServers, tomando cuidado para manter o JSON formatado corretamente. Você pode usar um validador online pra ter certeza que está tudo certo.
Pronto!
Só reiniciar o Claude Desktop, abrir o Obsidian, e começar a usar! Para mais detalhes, veja o nosso repositório do mcp-obsidian.
Inscreva-se no nosso canal no YouTube
