2. Ordenar_facturas
import os
import shutil
import re
os.chdir("Descargas")
carpeta_facturas = "Facturas"
os.makedirs(carpeta_facturas, exist_ok=True)
for archivo in os.listdir():
if archivo.endswith(".pdf") and archivo.startswith("Factura_"): # Solo pdf, solo que empiecen por Factura.
# Examinar, comprobar patrón y extraer:
match = re.match(r"Factura_(\w+)_\d+_\d+\.pdf", archivo)
if match:
tipo = match.group(1)
# "Agua", "Luz", "Gas", etc.
destino = os.path.join(carpeta_facturas, tipo)
os.makedirs(destino, exist_ok=True)
shutil.move(archivo, os.path.join(destino, archivo))
print(f"Movido {archivo} a {destino}/")
else:
print(f"ERROR: No coincide: {archivo}")
# SALIDA FINAL BONITA con recuento de facturas dentro de carpetas.
print(f"\n ¡SE HA COMPLETADO LA ORGANIZACIÓN SATISFACTORIAMENTE!")
if os.path.exists(carpeta_facturas):
for tipo in os.listdir(carpeta_facturas):
subcarpeta = os.path.join(carpeta_facturas, tipo)
if os.path.isdir(subcarpeta):
num_facturas = len([f for f in os.listdir(subcarpeta) if f.endswith('.pdf')])
print(f" {tipo}: {num_facturas} facturas")

Este programa en Python organiza automáticamente tus facturas PDF en carpetas según su tipo (Agua, Luz, Gas, etc.) dentro de la carpeta "Descargas".
¿Cómo funciona?
Tener Python 3.x instalado
No se necesitan librerías adicionales (usa módulos nativos de Python)
Estructura de archivos requerida:
Formato de nombre de archivo:
Factura_TIPO_NÚMERO_FECHA.pdf
Ejemplos válidos:
Factura_Agua_123_202401.pdf
Factura_Luz_456_202402.pdf
Factura_Gas_789_202403.pdf
¿Qué carpetas crea?
Esta es la estructura que se crea:
Descargas/
├── Facturas/
│
├── Agua/
│
│
├── Factura_Agua_123_202401.pdf
│
│
└── Factura_Agua_124_202402.pdf
│
├── Luz/
│
│
└── Factura_Luz_456_202402.pdf
│
└── Gas/
│
└── Factura_Gas_789_202403.pdf
¿Cuál es el procedimiento?
Crea carpeta principal "Facturas"
Identifica el tipo de factura mediante expresiones regulares
Crea subcarpetas por cada tipo (Agua, Luz, Gas, etc.)
Mueve archivos a sus carpetas correspondientes
Proporciona feedback de lo que está haciendo
Muestra resumen final con conteo de facturas por tipo
Ejemplo de salida:
Movido Factura_Agua_123_202401.pdf a Facturas/Agua/
Movido Factura_Luz_456_202402.pdf a Facturas/Luz/
ERROR: No coincide: Factura_invalida.pdf
¡SE HA COMPLETADO LA ORGANIZACIÓN SATISFACTORIAMENTE!
Agua: 2 facturas
Luz: 1 facturas
Gas: 3 facturas
NOTAS IMPORTANTES
Solo procesa archivos PDF que empiecen por "Factura_"
Los archivos con formato incorrecto se reportan como error
No elimina ningún archivo, solo los mueve de carpeta
Las carpetas se crean automáticamente si no existen
Módulos utilizados:
os - Operaciones del sistema y manejo de archivos
shutil - Movimiento de archivos entre carpetas
re - Expresiones regulares para identificar patrones
Modificar...
Cambiar carpeta de origen: Modifica os.chdir("Descargas")
Ajustar patrón de nombres: Edita la expresión regular r"Factura_(\w+)\d+\d+.pdf"
Cambiar nombre de carpeta principal: Modifica carpeta_facturas = "Facturas"
