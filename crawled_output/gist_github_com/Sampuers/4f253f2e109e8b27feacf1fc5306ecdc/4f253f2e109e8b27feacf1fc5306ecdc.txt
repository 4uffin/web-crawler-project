4. Organizar_fotos_fecha_creación
import os
from PIL import Image
from PIL.ExifTags import TAGS
from datetime import datetime
def ordenar_fotos():
carpeta_origen = os.path.dirname(__file__)
carpeta_destino = os.path.join(carpeta_origen, "Fotos_Ordenadas")
os.makedirs(carpeta_destino, exist_ok=True)
# Recorrer todos los archivos de la carpeta_origen que sean formato jpg, jpeg y png.
for archivo in os.listdir(carpeta_origen):
if archivo.lower().endswith((".jpg", ".jpeg", ".png")):
path_origen = os.path.join(carpeta_origen, archivo)
try:
with Image.open(path_origen) as img:
# Obtener fecha EXIF
exif = img.getexif()
fecha = None
# Buscar la fecha original de la creación del archivo
for tag, valor in exif.items():
if TAGS.get(tag) == "DateTimeOriginal":
fecha = datetime.strptime(valor, "%Y:%m:%d %H:%M:%S")
break
# Si no encuentra EXIF, usar fecha de creación del sistema
if not fecha:
fecha = datetime.fromtimestamp(os.path.getctime(path_origen))
nuevo_nombre = fecha.strftime("%Y-%m-%d_%H-%M-%S") + os.path.splitext(archivo)[1].lower()
path_destino = os.path.join(carpeta_destino, nuevo_nombre)
os.rename(path_origen, path_destino)
print(f" {archivo} → {nuevo_nombre}")
except Exception as e:
print(f" ERROR en {archivo}: {e}")
# SALIDA FINAL ----->
ordenar_fotos()
print("Se ha procesado correctamente")

Organiza automáticamente tus fotos (JPG, JPEG, PNG) en una carpeta ordenada por fecha y hora de creación, usando los metadatos EXIF cuando están disponibles.
Características principales
Ordena por fecha real (metadatos EXIF)
Respaldo inteligente (fecha del sistema si no hay EXIF)
Mantiene formato original (jpg, jpeg, png)
Renombrado (AAAA-MM-DD_HH-MM-SS.ext)
Requisitos previos
Instalar dependencias:
pip install Pillow
Formatos soportados:
JPG (.jpg, .jpeg)
PNG (.png)
¿Cómo funciona?
Busca metadatos EXIF (fecha original de la foto)
Si no encuentra EXIF → usa fecha de creación del archivo
Renombra con formato: AAAA-MM-DD_HH-MM-SS.ext
Mueve a carpeta Fotos_Ordenadas
Ejemplo de renombrado
Foto1.jpg
→ 2023-08-15_14-30-25.jpg
IMG_1234.png
→ 2024-01-20_09-15-42.png
vacaciones.jpeg
→ 2024-03-10_18-45-13.jpeg
IMPORTANTE sobre EXIF
¿Qué es EXIF?
EXIF (Exchangeable Image File Format) es un estándar de metadatos que las cámaras digitales y teléfonos añaden automáticamente a cada foto que tomas.
¿Qué información contiene EXIF?
Fecha y hora exacta de la foto
Marca y modelo de la cámara/teléfono
Configuración técnica: apertura, ISO, velocidad de obturación
GPS (ubicación donde se tomó la foto)
Orientación (horizontal/vertical)
Y mucho más...
¿Cuándo puede faltar EXIF?
Fotos editadas con programas que borran metadatos
Capturas de pantalla
Fotos descargadas de internet
Imágenes muy antiguas
Módulos utilizados
PIL (Pillow) - Procesamiento de imágenes y EXIF
os - Operaciones de archivo y carpetas
datetime - Manejo de fechas y horas
IMPORTANTE
El script mueve archivos, no copia
Haz backup antes de usar con fotos importantes
Revisa errores para fotos problemáticas
Personalización
Cambiar formato de fecha
Añadir más formatos
